# Apple-I の歴史的価値

## 1. 誕生の経緯

1976年、Steve Wozniak（Woz）はホームブリュー・コンピュータクラブ（HOMEBREW Computer Club）に
単独で設計した回路基板を持ち込んだ。当時のホビイストは紙テープやトグルスイッチで
機械語を直接入力するのが当たり前だったが、Woz が見せたものは違った。
**キーボードで文字を打つとそのまま画面に表示される** — それだけのことが、
会場にいた全員を驚かせた。

これが Apple-I である。価格は $666.66（約 20 万円相当）、手作り基板が 200 枚ほど出荷された。

---

## 2. 技術的な革新

### 2.1 チップ選定の妙

Woz は MOS Technology **6502**（$25）を採用した。
当時の主流だった Intel 8080（$179）や Motorola 6800（$175）より 1 桁安い。
単なるコスト削減ではなく、6502 のアーキテクチャがよりエレガントで
少ない命令数でプログラムを書けると確信していた。

| CPU | 価格 | アドレッシングモード | 採用機 |
|-----|------|---------------------|--------|
| Intel 8080 | $179 | 5 種 | Altair 8800 |
| Motorola 6800 | $175 | 7 種 | SWTPC 6800 |
| MOS 6502 | $25 | 13 種 | Apple-I / II, Commodore 64, Famicom |

### 2.2 最小構成の美しさ

Apple-I の主要チップはわずか 3 種類：

```text
6502         ← CPU
6821 PIA     ← キーボード・ディスプレイ I/O
2513 ROM     ← キャラクタジェネレータ
（+ SRAM、ビデオ回路）
```

Woz Monitor（256 バイト）だけで 16 進ダンプ・書き込み・実行の
すべてが完結する。ソフトウェアとハードウェアの最小性が徹底されている。

### 2.3 Woz Monitor の設計思想

`$FF00–$FFFF` の 256 バイトに収まったモニタプログラムは、
バイト単位の直接操作という「コンピュータとの生の対話」を提供した。

```text
FF00: D8        ← メモリの中身を見る
0300: A9 55 4C 00 03   ← 機械語を書く
0300R           ← 実行する
```

GUIもOSも不要。アドレスとデータだけで完全に制御できる。

## 3. Apple Computer 社の誕生

Woz の親友 Steve Jobs はこの基板に商業的な価値を見た。
1976年4月1日、**Apple Computer** が設立される。

- 最初の注文: Byte Shop（コンピュータ専門店）から 50 台
- 売上でコンポーネント代を賄い、Apple II の開発資金を作った
- Apple II（1977）が大ヒットし、パーソナルコンピュータ産業が離陸した

Apple-I は Woz が「友人に見せたかった」だけのものが、
偶然に世界最大のテクノロジー企業の出発点になった。

---

## 4. Integer BASIC の意味

カセットテープで配布された Integer BASIC（$E000–$EFFF、4 KB）は
Woz が独力で書いたインタープリタである。

- **浮動小数点なし**（-32768〜32767 の整数のみ）
- **4 KB 以下に完全収録**
- Apple II Integer BASIC の直接の前身

「PRINT 2+2」と打って「4」が返ってくること。
それが 1976 年には革命的な体験だった。

## 5. 現在における希少価値

オリジナルの Apple-I 基板は現存数が非常に少ない（推定 60〜70 台）。

| 年 | オークション落札額 |
|----|-----------------|
| 2012 | $374,500（約 3,000 万円） |
| 2014 | $905,000（約 9,500 万円） |
| 2022 | $400,000（約 5,500 万円） |

動作品はさらに希少で、博物館や個人コレクターが保存している。

## 6. soft-FPGA でシミュレートする意義

命令エミュレータ（例: MAME）は Apple-I を動かすことができる。
しかし、ソフトウェアエミュレータには **見えないもの** がある。

| 観測できる情報 | エミュレータ | soft-FPGA (Verilator) |
|--------------|-------------|----------------------|
| レジスタ (A, X, Y, SP, PC, P) | ✓ | ✓ |
| メモリ内容 | ✓ | ✓ |
| バスサイクル (AB / DB / WE) | ✗ | **✓** |
| T ステート（サイクル単位の信号遷移） | ✗ | **✓** |
| PIA ハンドシェイク (CA1 / CB1 ストローブ) | ✗ | **✓** |
| ページクロスによる +1 サイクルペナルティ | ✗ | **✓** |
| RMW 命令の無効ライトサイクル | ✗ | **✓** |

RTL シミュレーションは「回路が動いている」のを見ることができる唯一の手段である。
Apple-I のシンプルな構造（RAM 4 KB / ROM 4.25 KB / PIA 1 個）は、
この可視化プラットフォームの最初のターゲットとして理想的だった。
