# 13. テスト手順

`examples/06-8080/build/cpm` が提供するテストモードの実行手順と期待出力。

## 1. ビルド

```bash
cd examples/06-8080
cmake -B build
cmake --build build -j$(nproc)
# → build/cpm が生成される
```

## 2. テストモード一覧

| コマンド | 内容 | 期待出力 |
|---------|------|---------|
| `cpm --test` | ハーネス内蔵スモークテスト | `[test] sim_test: 2 bytes out (Hi)` |
| `cpm --boot-test` | CP/M ブートテスト | `--- CP/M boot: OK (A> found) ---` |
| `cpm --run-test test/test_all.bin` | 命令テストスイート (20グループ) | `PPPPPPPPPPPPPPPPPPPP` |
| `cpm --exec 8080EX1` | CPU エクササイザ (20グループ) | `Tests complete` + exit 0 |

ファイルパスはバイナリの場所から自動解決されるため、**どのディレクトリから実行してもよい**。

## 3. 各テストの詳細

### 3.1 スモークテスト

vm80a 単体の最小確認。CP/M 不要。

```bash
./build/cpm --test
```

```text
[test] sim_test: 2 bytes out (Hi)
```

exit 0 = PASS。内部で `MVI A,'H' / OUT 01 / MVI A,'i' / OUT 01 / HLT` を実行し、
CONOUT に 2 文字以上出力されれば合格。

### 3.2 CP/M ブートテスト

BIOS → CCP+BDOS の起動シーケンスを自動検証する。

```bash
./build/cpm --boot-test
```

```text
A>
--- CP/M boot: OK (A> found) ---
```

`A>` プロンプトが現れれば exit 0。現れなければ exit 1。

### 3.3 命令テストスイート

CP/M なしで動く 20 グループの命令テスト。事前にアセンブルが必要。

```bash
cd examples/06-8080
make -C test
./build/cpm --run-test test/test_all.bin
```

```text
PPPPPPPPPPPPPPPPPPPP
--- 20 group(s) tested: ALL PASS ---
```

各グループ `P` = PASS、`F` = FAIL。exit 0 = 全グループ PASS。

| グループ | 対象命令 |
|---------|---------|
| G01 | MVI (LD r,n) |
| G02 | MOV (LD r,r) |
| G03 | LXI / PUSH / POP |
| G04 | XCHG / XTHL / PCHL / SPHL |
| G05 | ADD |
| G06 | ADC |
| G07 | SUB |
| G08 | SBC |
| G09 | AND |
| G10 | OR / XOR |
| G11 | CMP |
| G12 | INC |
| G13 | DEC |
| G14 | INX / DCX / DAD |
| G15 | STA / LDA / STAX / LDAX / SHLD / LHLD |
| G16 | RLCA / RRCA / RLA / RRA / SCF / CCF / CPL / DAA / NOP / DI / EI |
| G17 | JP（全8条件） |
| G18 | CALL（全8条件） |
| G19 | RET（全8条件） |
| G20 | RST 1 / RST 7 |

### 3.4 CPU エクササイザ

CP/M 上で 8080EX1.COM を実行する網羅テスト。

```bash
./build/cpm --exec 8080EX1
```

```text
A>8080EX1
8080 instruction exerciser
ld r,n (MVI)....................OK
ld r,r (MOV)....................OK
lxi/push/pop....................OK
ex/xthl/pchl/sphl...............OK
add a,r.........................OK
adc a,r.........................OK
sub r...........................OK
sbc a,r.........................OK
and r...........................OK
or/xor r........................OK
cp r............................OK
inc r...........................OK
dec r...........................OK
inx/dcx/dad.....................OK
```

全グループ `OK` で「Tests complete」が出力され exit 0 = PASS。

## 4. 全テスト一括実行

```bash
cd examples/06-8080
make -C test                         # test_all.bin を生成
./build/cpm --test                   # スモーク
./build/cpm --boot-test              # ブート
./build/cpm --run-test test/test_all.bin  # 命令スイート
./build/cpm --exec 8080EX1           # CPU エクササイザ
```
