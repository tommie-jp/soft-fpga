# 01. 8080 / CP/M 実装計画

## 1. 概要

vm80a（Intel 8080A デキャップ起こし RTL）を Verilator で C++ 化し、
CP/M 2.2 を動作させてブラウザから xterm.js でターミナル操作できるようにする。

## 2. アーキテクチャ

![06-8080 アーキテクチャ](../../docs/img/06-8080-arch.png)

## 3. メモリマップ

| アドレス | 用途 |
|---------|------|
| $0000–$00FF | ページ 0（リスタートベクタ・BDOS エントリ） |
| $0100–$DBFF | TPA（ユーザプログラム領域） |
| $DC00–$E3FF | CCP（コマンドラインプロセッサ、2 KB） |
| $E400–$F1FF | BDOS（基本ディスク OS、3.5 KB） |
| $F200–$FFFF | BIOS（カスタム実装、最大 3.5 KB） |

## 4. RTL コア — vm80a

- リポジトリ: `https://github.com/1801BM1/vm80a`
- NEC D8080AFC のデキャップ画像から起こされた Verilog RTL
- バグ互換・タイミング互換・割り込み応答が実機と一致
- `examples/06-8080/verilog/vm80a/` にサブモジュールとして配置

## 5. BIOS 実装

`examples/06-8080/sw/cpm/bios/bios.asm` に実装済み。  
IN/OUT ポート命令で C++ ハーネスと通信する。

| 機能 | 状態 |
|------|------|
| BOOT/WBOOT | ✅ 実装済み |
| CONIN/CONOUT/CONST | ✅ 実装済み |
| SELDSK/SETTRK/SETSEC/SETDMA | ✅ 実装済み |
| READ/WRITE | ✅ 実装済み |
| SECTRAN | ✅ 実装済み（1:1） |
