# 52. メモリマップ

## 1. CP/M 2.2 標準メモリ配置（64 KB）

| アドレス | サイズ | 用途 |
|---------|--------|------|
| $0000–$00FF | 256 B | ページ 0（リスタートベクタ・BDOS エントリ） |
| $0100–$DBFF | 55 KB | TPA（Transient Program Area） |
| $DC00–$E3FF | 2 KB | CCP（Console Command Processor） |
| $E400–$F1FF | 3.5 KB | BDOS（Basic Disk Operating System） |
| $F200–$FFFF | 3.5 KB | BIOS（Basic I/O System） |

## 2. ページ 0 詳細

| アドレス | 内容 |
|---------|------|
| $0000–$0002 | `JMP WBOOT`（ウォームブートベクタ） |
| $0003 | IOBYTE（I/O 割り当て、オプション） |
| $0004 | 現在のディスク番号 |
| $0005–$0007 | `JMP BDOS+6`（BDOS エントリポイント） |
| $0008–$003F | リスタートベクタ RST1–RST7（8080 割り込み） |
| $0040–$00FF | スクラッチエリア |

## 3. I/O ポートマップ

| ポート | 方向 | 用途 |
|--------|------|------|
| $00 | IN | CONIN: コンソール入力 |
| $01 | OUT | CONOUT: コンソール出力 |
| $02 | IN | CONST: コンソールステータス |
| $10 | OUT/IN | DCMD/DSTS: ディスクコマンド / 結果 |
| $11 | OUT | DTRK: トラック番号 |
| $12 | OUT | DSEC: セクタ番号 |
| $13 | OUT | DDMA\_L: DMA アドレス低バイト |
| $14 | OUT | DDMA\_H: DMA アドレス高バイト |

## 4. ディスクパラメータ（標準 8 インチ SSSD）

| 項目 | 値 | 計算式 |
|------|----|-------|
| SPT（セクタ/トラック） | 26 | — |
| セクタサイズ | 128 B | — |
| トラック数 | 77 | — |
| ブロックサイズ | 1 KB | 2^BSH × 128 B |
| 総容量 | 256 KB | 77 × 26 × 128 |
| 予約トラック（OFF） | 2 | システム領域 |
| 有効容量 | 243 KB | (77-2) × 26 × 128 |
| ディレクトリエントリ | 64 | DRM+1 |
